HS 编码即海关编码，全称为《商品名称及编码协调制度》编码，它是对各种不同产品进行分类的一种标准化编码系统。HS 编码由数字组成，用于确定进出口货物的类别，不同的编码对应不同的商品种类，方便海关对货物进行统计、监管和征税等工作。
## 主流HS编码查询网站

| 名称 | 网址 | 简介 |
| --- | --- | --- |
| 中国海关网站 | [http://www.customs.gov.cn](http://www.customs.gov.cn/) | 中国海关总署官方网站，可查询最新版本的中国海关HS编码，还提供丰富的贸易数据和服务 |
| 中国海关总署HS编码查询系统 | http://www.customs.gov.cn/publish/portal0/tab5195/ | 中国海关总署官方提供的HS编码查询系统，可根据商品名称、国际商品分类法（HS）编码、海关税则号查询中国海关HS编码 |
| 全球海关编码查询网 | [https://www.trade-comm.cn](https://www.trade-comm.cn/) | 可查询中国海关HS编码以及全球其他国家的海关HS编码 |
| 中国海关进出口商品编码表 | [https://www.tjhscode.com](https://www.tjhscode.com/) | 中国海关官方网站发布的进出口商品编码表，可查询最新版本的中国海关HS编码 |
| 中国海关进出口商品编码查询系统 | [http://www.hs-codes.net](http://www.hs-codes.net/) | 中国海关总署官方提供的进出口商品编码查询系统，可查询最新版本的中国海关HS编码 |

| 税号查询网 | [http://www.taxtools.com.cn](http://www.taxtools.com.cn/) | 可以查询中国海关HS编码，也可以查询国际海关HS编码 |
